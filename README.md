# 십일전략 가맹 모집 | 10days-franchise

GR831 약술형논술 연구소의 공식 온라인 가맹 모집 B2B 사이트.
학원 원장님 대상 가맹 안내 페이지입니다.

## 기술 스택

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Noto Serif KR + Pretendard** (한글 폰트)

## 반응형

- 모바일 / 태블릿 / PC 대응
- PC 사용 비율이 높은 B2B 타겟

## 실행 방법

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 빌드

```bash
npx next build
```

## 배포

Vercel을 통해 자동 배포됩니다.
